# 稳稳睡觉啦白噪音 - 微信小程序

帮宝宝入睡的白噪音微信小程序，内置多种白噪音，支持定时停止、收藏功能和哄睡记录。

## ✨ 功能

- 🌧️ **17种白噪音**，涵盖自然声、环境声、轻音乐、B超扫描声
- ⏱️ **定时自动停止**，支持自定义时长
- ⭐ **收藏功能**，收藏喜欢的声音
- 📋 **哄睡记录**，自动记录宝宝入睡时间和哄睡时长
- 📊 **数据统计**，查看平均入睡时间、最快哄睡记录、声音排名
- 🔄 **一键分享**，把实用的哄睡声音分享给其他宝妈
- 🎤 **用户录音**，支持录制自己的哄睡声
- 📱 **完全离线使用**，无需网络
- 💾 **所有数据存在本地**，不涉及隐私

## 📱 页面结构

```
├── 首页 - 声音列表 + 播放控制 + 哄睡记录
├── 我的录音 - 用户自定义录音
└── 我的收藏 - 收藏的声音 + 哄睡记录入口
    └── 📋 我的哄睡日记 - 历史记录列表
        └── 📊 本月哄睡统计 - 数据分析
```

## 🛠️ 导入项目

1. 打开 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 选择"导入项目"
3. 选择 `baby-white-noise` 文件夹
4. 输入你的小程序 AppID
5. 导入成功后，你需要：
   - 将真实的 MP3 音频文件放到 `assets/audio/` 目录下替换占位文件
   - 在微信开发者工具中检查"基本信息"-"分包大小"确认不超过限制
6. 提交审核

## 💡 使用说明

### 哄睡记录功能

1. 点击任意声音开始播放
2. 在弹出的对话框中选择"开始记录"
3. 播放栏上方会显示实时计时 🛌 哄睡中...
4. 宝宝睡着后点击"停止"按钮
5. 自动弹出哄睡完成总结，记录入睡时间和时长
6. 在「我的收藏」→「我的哄睡日记」查看历史记录和统计数据

### 数据统计项

- 平均入睡时间
- 最早入睡记录
- 最近一周睡眠趋势
- 平均哄睡时长
- 最快哄睡记录
- 🏆 哄睡最快声音排名 Top 5

## 🔒 隐私说明

所有数据（包括哄睡记录、收藏的声音等）都只保存在用户手机本地，不会上传到任何服务器，完全保护隐私。
